# Template 09 — Certificate of Authenticity

A one-page printable certificate that ships with each serialized instrument. Frame-able. Numbered. Signed.

## Format

- **One letter page, portrait, often printed on heavier cream stock.**
- **Top third:** wordmark + mark + "Certificate of Authenticity" header in display italic serif.
- **Middle third:** the certified facts — *"This certifies that the instrument identified below…"*. Block: serial, type, family, build date, materials, key/tuning, edition (e.g., "1 of 3 in the magazine-baseline series").
- **Bottom third:** signature line with Tony's printed name and "Tony Koop, Heifer Zephyr · Santa Clara, CA" + optional embossed mark area.
- **Border:** a thin double-rule frame in family-accent.

## Files this template consumes

| Source | Used for |
|---|---|
| `design.md` | Serial, type, materials, build date |
| `validation.csv` | Tuning summary line (range and tolerance) |
| `brand/assets/signature.svg` | Tony's signature image |

## Voice

Formal but spare. Three sentences. No marketing.

## What "done" looks like

- Serial appears at least twice (header + middle block).
- Edition is stated honestly — *"1 of 3 in the planned key-tuning DoE series"* if applicable, or *"unique"* if it's a one-off.
- Signature is rasterized from `brand/assets/signature.svg`.
- Prints cleanly on cream stock without color shift.
- Optional QR code linking to the sales page (Template 04) or sale-history record.
