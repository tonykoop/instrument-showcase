# Template 15 — Catalog / Lookbook (Multi-Instrument)

A magazine-style multi-instrument catalog. The flagship marketing artifact — what you put on the booth table at a fair, what you mail to a gallery, what you pin to the front of the maker site.

## Format

- **Multi-page, letter portrait, web + PDF (and ideally print-ready PDF for short-run printing).**
- **Spread layout:** alternating image-forward and text-forward spreads.
- **Sections:**
  1. **Cover** — wordmark, mark, season/year, hero composite photo.
  2. **Foreword** — half-page essay from the maker (2 paragraphs distilled from the bio).
  3. **The collection** — 6–12 instruments, each on a single page or 2-page spread:
     - Hero photo
     - Instrument name + serial-series prefix
     - 2–3 sentence description
     - Spec excerpt (4–6 lines)
     - Repo / sales-page URL
  4. **Behind the work** — 2-page spread of process photos (workshop, in-progress, materials).
  5. **About the maker** — full bio + patent + teaching.
  6. **Index + contact + colophon.**

## Files this template consumes

This template aggregates content across **many** instrument folders.

| Source | Used for |
|---|---|
| Each instrument folder's `README.md` + `images/` | Per-instrument page |
| Each instrument's `design.md` | Spec excerpt |
| `tonykoop/README.md` | Foreword + maker bio |
| `retail-displays/images/` (if shown) | Process / behind-the-work |
| User-curated selection | Which instruments are "in the collection" this season |

## Voice

Editorial-magazine. Foreword is the longest piece of prose in the entire instrument-showcase skill. Per-instrument descriptions are the brochure tagline + one extra sentence.

## What "done" looks like

- Reads cover-to-cover in 8–12 minutes.
- 6–12 instruments featured, no more.
- Every instrument page has its repo URL.
- Process spread is real, not stock.
- Bound or saddle-stitched at print, single-PDF for digital.
