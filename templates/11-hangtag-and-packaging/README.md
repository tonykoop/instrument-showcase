# Template 11 — Hangtag, Box Label, and Packaging Insert

The retail-ready printables that physically attach to the instrument or its packaging. Three sub-formats in one template (because they share the same content):

- **Hangtag** — small die-cut tag tied to the instrument with twine.
- **Box label** — adhesive label affixed to the shipping box / case.
- **Packaging insert** — small slip placed inside the box, includes a "first 5 minutes" guide.

## Format

- **Hangtag:** 2.5 × 4 in, hole at top, double-sided.
- **Box label:** 4 × 6 in, adhesive, single-sided.
- **Packaging insert:** 5 × 7 in, double-sided.

All printed N-up on letter sheets with crop marks, ready for cutting on the workshop laser or trimmer.

## Files this template consumes

| Source | Used for |
|---|---|
| `design.md` | Serial, type, key, variant |
| `bom.csv` | Materials one-liner |
| `validation.csv` | Tuning tolerance one-liner (insert only) |
| `assembly-manual.md` "first play" notes if any | Packaging insert "first 5 minutes" |

## Voice

Hangtag/label: telegraphic. Insert: warm imperative ("Pick it up. Strike it lightly with the included mallet.").

## What "done" looks like

- Hangtag prints 6-up on letter with crop marks.
- Box label has serial + recipient + return address slots, with cling for shipping.
- Insert is folded once and fits an instrument-sized box.
