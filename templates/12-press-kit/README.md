# Template 12 — EPK / Press Kit One-Pager

For media, galleries, festival booking, and writers asking for a press resource. One letter page, plus a downloadable hi-res asset bundle.

## Format

- **One letter page, portrait.**
- **Top:** wordmark + mark + "Press Kit" label.
- **Maker bio (3 paragraphs):** distilled from the `tonykoop` profile README — engineering, instruments, teaching.
- **Selected works:** 4–6 instruments with thumbnail + 1-line description + repo link.
- **Sound samples:** links to recorded clips (if available).
- **Awards / patents / press mentions:** including US Patent 11,137,017 B2.
- **Contact + booking line.**
- **Asset bundle reference:** a separate folder/zip with hi-res photos and bio text.

## Files this template consumes

| Source | Used for |
|---|---|
| `tonykoop/README.md` | Maker bio + portfolio links |
| Multiple instrument folders' `README.md` | Selected-works thumbnails and 1-liners |
| `images/` across instrument folders | Photo bundle |
| `tonykoop/docs/` (if present) | Press mentions, awards |

## Voice

Quote-friendly. Every paragraph in the maker bio could be lifted as a press quote and stand alone. No marketing puffery — the patents and the work do the lifting.

## What "done" looks like

- Reads as the canonical "about Tony" resource for any outside writer.
- Selected works are accurate (not a stale list — pull live from current portfolio).
- Includes the patent number with link.
- Asset bundle path is real and contains the referenced photos.
